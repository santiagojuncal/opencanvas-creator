## 🇪🇸 Español

### 🚀 Descripción

OpenCanvas es un editor visual que permite crear landing pages modernas sin necesidad de escribir código.

Proporciona una interfaz drag-and-drop intuitiva para diseñar páginas y exportarlas como HTML y CSS listos para producción, sin depender de backend ni base de datos.

---

### ✨ Funcionalidades

* Editor visual drag-and-drop
* Componentes básicos (texto, imágenes, botones, secciones, formularios)
* Diseño responsive automático
* Personalización de estilos (colores, tipografías, layout)
* Exportación de código limpio (HTML + CSS)

## 🇺🇸 English

### 🚀 Description

OpenCanvas is a visual editor that allows users to create modern landing pages without writing code.

It provides an intuitive drag-and-drop interface and exports production-ready HTML and CSS, with no backend or database required.

---

### ✨ Features

* Drag-and-drop visual editor
* Basic components (text, images, buttons, sections, forms)
* Automatic responsive design
* Style customization (colors, fonts, layout)
* Clean code export (HTML + CSS)
